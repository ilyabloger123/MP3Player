# 🎵 MP3Player by ilyabloger123 

Современный и интерактивный аудио-плеер на C#, созданный с использованием движка **BASS.net**. Это не просто проигрыватель, а проект с глубокой кастомизацией интерфейса и скрытыми игровыми механиками.

## 🚀 Основные возможности
* **Профессиональный звук:** Полноценная поддержка MP3 и WAV через библиотеку BASS.
* **Кастомный UI:** Полностью безрамочный дизайн (`FormBorderStyle.None`) со скругленными углами и поддержкой темной/светлой тем.
* **Визуализация:** Стерео VU-метры (визуализатор), работающие в реальном времени.
* **Эффекты:** Управление темпом (Fast Mode), балансом (Pan) и плавное затухание звука при паузе.
* **Интерактивность:** Весь интерфейс реагирует на музыку — заголовок и кнопки «подмигивают» в такт басам.

## 🕹 Пасхалки и Секреты
Проект содержит уникальные интерактивные элементы:
1. **Destruction Mode:** Если кликнуть по заголовку 10 раз, он «отвалится», плеер покроется трещинами, а звук начнет «умирать» с искажениями

## 🛠 Технологический стек
* **Язык:** C# (.NET Framework)
* **Библиотеки:** [BASS.net](http://un4seen.com), [TagLib](https://www.nuget.org/packages/TagLibSharp/) — обработка аудио, GDI+ — кастомная отрисовка эффектов, отображение обложки .
* **Инструменты:** Visual Studio Installer Projects.

## 📦 Установка
1. [📥 **Скачать установщик MP3Player (.msi)**](https://github.com/ilyabloger123/MP3Player/releases/download/v1.1/MP3Player_setup.msi)
2. Запустите установщик и следуйте инструкциям.
